# 고급 AI 융합프로젝트 
전남대학교 인공지능융합학과 고급 AI융합프로젝트 결과물입니다.

2D rgb이미지로부터 smpl joints 추정하여 유니티로 시각화하고, 잘못 추정된 joint를 사람이 수정하여 저장할 수 있는 
3차원 human joint annotion tool 입니다.

1. 사용언어
   - C# (Unity)
   - python
2. 사용 기술
   - CLIFF https://github.com/haofanwang/CLIFF
